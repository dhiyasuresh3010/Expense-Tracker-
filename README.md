## 🧾 Project Description

The Expense Tracker is a Python-based command-line application that helps users record, manage, and analyze their daily expenses efficiently. It stores data locally using JSON and provides multiple features such as filtering, summaries, sorting, and CSV export.            
This project is designed for learning Python file handling, data structures, and basic data analysis concepts.

## 🚀 Features
➕ Add new expenses (date, category, amount, description)            
📋 View all expenses in a structured format                      
✏️ Edit existing expense records                    
❌ Delete unwanted expenses                    
💰 Calculate total expenses                    
🔍 Search expenses by category                      
📊 Category-wise expense summary                    
🥇 Identify highest expense                        
📅 Monthly expense report                     
🔄 Sort expenses by amount or date                         
📤 Export data to CSV file for Excel use                   
💾 Persistent storage using JSON file
